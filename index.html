Veron1CA

About

Veron1CA is an open source Discord moderation & music bot designed to do server management and entertain both at once. It has a bunch of easy-to-use commands that can pretty much make the work of customizating your Discord server a breeze!

Features

Now this is where things get interesting...

Veron1CA is the combination of a Discord moderation bot and a music bot. It comes packed with features like built-in swear filter, message author jailing abilities and so on. Along with a lot of server management and modding capabilities, it can play music too! Useful features like shuffle, loop are also available with it. If you want to increase your experience with swear filter, Veron1CA also comes with an expandible database of unwanted words.

Updates

And speaking of updates...

The source code for Veron1CA is being frequently monitored and the bot itself is being tested several times a day. So whenever an issue comes in, we try to fix it as soon as possible. Also, the source code for the bot is open source, so anyone who's willing to contribute to the project can go ahead and make their own changes!

Commands

So, you wanna know how to access Veron1CA?

To control Veron1CA, you need three special roles, BotPilot, BotMod and BotAdmin. BotPilot is for standard operations like sending DMs, playing music, greeting members etc. On the other hand, BotMod and BotAdmin - these two are used to access server management and moderation tools/commands. When you have atleast one of these three roles assigned to you, you can start by typing //help in chat. The default command prefix for Veron1CA is // but as it's open source, you can change it anytime!
