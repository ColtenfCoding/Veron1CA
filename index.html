Veron1CA

About

Veron1CA is an open source Discord moderation & music bot designed to do server management and entertain both at once. It has a bunch of easy-to-use commands that can pretty much make the work of customizating your Discord server a breeze!

Features

Now this is where things get interesting...

Veron1CA is the combination of a Discord moderation bot and a music bot. It comes packed with features like built-in swear filter, jailing and so on. Along with a lot of server management and modding capabilities, it can play music too! Useful features like shuffle, loop are also available with it. If you want to increase your experience with swear filter, Veron1CA also comes with an expandible database of unwanted words.

Updates

And speaking of updates...

The source code for Veron1CA is being frequently monitored and the bot itself is being tested several times a day. So whenever an issue comes in, we try to fix it as soon as possible. Also, the source code for the bot is open source, so anyone who's willing to contribute to the project can go ahead and make their own changes!

Commands

So, you wanna know how to access Veron1CA?

There are 3 default roles configured to access the bot - BotPilot for normies, BotMod for server moderators and BotAdmin for server admins. Once you have one of these three roles assigned to yourself, you can go ahead and ping @Veronica in chat or use the traditional //help command to see what it can do!
